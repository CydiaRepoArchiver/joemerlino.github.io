Pebble Profiles

This tweak has two primary features:
1. It can prevent an iPhone from sending notifications to a Pebble when the phone is unlocked.
2. It can filter out which applications can send notifications to the Pebble.
Enable/disable the tweak, enable/disable Pebble DnD feature in settings or using the dedicated flipswitch toggles. You can also Whitelist the enabled apps in pebble dnd mode to receive them anyway.

Download it on Cydia

StopAutoDelete

Stops the cursor from deleting entire words when holding the backspace button for a long time. Enable/disable in settings or using the dedicated flipswitch toggle.

Download it on Cydia

DarkCC

Makes the Control Center darker. Enable/disable in settings or using the dedicated flipswitch toggle.

Download it on Cydia

AirplaneSignal

Turns on AirplaneMode if you have low signal strength. Enable/disable in settings or using the dedicated flipswitch toggle. Set the strength limit (below 20% - 40% or 60%). You can also set an activation delay and force the wifi to turn on after Airplane Mode. Plus you can re-check the signal after a set period of time.

Download it on Cydia

TelegramPlayback

Keeps playing audios even if you open other applications. You can also open other Telegram conversations while listening if you enable the switch inside the dedicated preference pane.

Download it on Cydia

MessagesTint

Tint conversations navigation bar with latest message status. The colors are green for sms, blue for iMessage, red if message hasn't been sent and orange if you have a draft message. Turn it on/off, set square bubbles and color the keyboard same as the navigation bar from the preference pane.

Download it on Cydia
